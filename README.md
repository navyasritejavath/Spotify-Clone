# Music Player

This project is a simple Spotify-inspired music player built using HTML, CSS, and JavaScript. The main idea behind this project was to understand how a real music player works and to practice core front-end development concepts such as DOM manipulation, event handling, and working with audio in JavaScript.

While it is not a full clone of Spotify, it recreates some of the basic and important features like playing songs, pausing, switching tracks, and displaying the currently playing song. The focus of this project was more on functionality and learning rather than creating a perfect UI.

## About the Project

The application has a clean and simple layout. At the top, there is a navigation bar with a logo and menu options. In the center, there is a song list section where multiple songs are displayed along with their cover images and play buttons. At the bottom, there is a music control panel which includes play, pause, next, and previous buttons, along with a progress bar.

Each song in the list can be clicked and played individually. When a song starts playing, the UI updates automatically to show the current song name, and a small animation appears to indicate playback.

## Features

Some of the main features included in this project are:

* Play and pause music using the main control button
* Play songs directly from the song list
* Switch between songs using next and previous buttons
* Display the currently playing song name
* Progress bar that updates as the song plays
* Ability to seek (move forward or backward) in the song
* Animated indicator that appears when music is playing
* Simple and clean user interface

## Technologies Used

This project is built using basic web technologies:

* HTML for structuring the webpage
* CSS for styling and layout
* JavaScript for adding interactivity and handling audio playback
* Font Awesome for icons

No frameworks or external libraries were used, which helped in understanding the fundamentals clearly.

## How It Works

The core of the project is the JavaScript Audio object. Whenever a user clicks on a song or presses a control button:

* The selected audio file is loaded
* The song starts playing or pauses depending on the action
* The UI updates, including icon changes, song name updates, and animation
* The progress bar moves as the song plays

Event listeners are used to detect user actions like clicks and input changes, and the interface is updated accordingly.

## Project Structure

The project is organized in a simple way:

* index.html – Contains the structure of the application
* style.css – Handles all the styling
* script.js – Contains the logic for music playback
* music/ – Folder containing audio files
* Image files – Used for logo, background, and song covers
